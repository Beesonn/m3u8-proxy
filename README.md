# M3U8 Proxy Server

A lightweight proxy server for HLS M3U8 playlists with Origin/Referer spoofing.

## Quick Deploy

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Beesonn/m3u8-proxy)

## Usage

`https://your-vercel-url.vercel.app/?url=https://example.com/playlist.m3u8`

`https://your-vercel-url.vercel.app/?url=https://example.com/playlist.m3u8&origin=https://megaplay.buzz&referer=https://megaplay.buzz/`
